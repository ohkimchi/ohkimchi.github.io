2018-04-05
#Work

曾几何时，我生命中除了我爸于我最重要的男人对我说，如果你真的要开始学一个库（library），你就去看它的官网去查它的原码，这是最好的学习方法。图样的我对他说的话习惯性的忽略了，直到我开始重新拾起Git。Git官网里有最详细最简洁最全面最好看的关于Git的使用指导，所以我想特别提醒小白不要浪费这最重要的资源而轻信旁门左道，沉迷stackoverflow各种散乱的小问题而忽略了系统性的思维。曾经有一篇心理学研究报告表明，散乱的知道点只能引起大脑皮层海马区（待检验，欢迎纠正）一小小小区域的兴奋，而系统性的思维和了解可以引起大脑皮层一大大大大片的刺激，幸福感更强。想要幸福，筒子们，系统学习才是王道。已扯远，现在拉回来。
